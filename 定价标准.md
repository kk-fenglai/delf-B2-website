# DELFluent 定价标准（EUR 锚定）

> 生效日期：2026-06-05  
> 锚定货币：**EUR**（与 `STRIPE_ANCHOR_CURRENCY=EUR`、Embedded Checkout + Adaptive Pricing 一致）  
> 代码来源：`backend/src/constants/pricing.js` · 运营可在管理后台「商品价目」微调

## 5. 定价原则（红线）

1. **EUR 为唯一锚定** — 改价先改 EUR，再按季度 FX 刷新 USD/CNY 参考价。
2. **毛利底线 ≥ 80%** — 按「满配额 + Stripe + 基础设施」测算，三档月付均达标。
3. **年付 = 10 个月** — 不叠加深折扣，避免侵蚀 MRR。
4. **FREE 不给 AI** — 仅练习配额；AI 能力通过试用 3 天或付费解锁。
5. **regrade 计入配额** — 防刷 token。
6. **日限不可取消**（AI 无限档）— 防 API 滥用。

## 6. 运维

```bash
# 首次或改价后同步 DB
cd backend && npm run seed:billing

# 环境变量
STRIPE_ANCHOR_CURRENCY=EUR
STRIPE_ADAPTIVE_PRICING=true   # 默认开启
STRIPE_CHECKOUT_UI=embedded
```

管理后台 → **支付管理 → 价格审阅**：查看 USD/CNY 相对 EUR 锚定的偏离；**≥10%** 建议调价。

**价目唯一性**：每个商品下，同一 **周期（months）+ 币种（currency）** 只能有一条价格（如 `STANDARD_12M` EUR 与 `STANDARD_12M_EUR` 不可并存）。重复条目请在后台停用后永久删除。

Stripe Dashboard 需为各 EUR 订阅档配置对应 `stripePriceId`（`Price` 表字段）。
